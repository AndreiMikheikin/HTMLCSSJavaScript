Выбрал расширение png т.к.:
1. изображение на однородном фоне,
2. изображение содержит больше 256 цветов,
3. изображение можно использовать с прозрачным фоном,
4. не удастся уменьшить объем файла, путем преобразования в другой формат, без потери качества.

Выбрал расширение jpeg т.к.:
1. изображение на неоднородном фоне,
2. изображение содержит больше 256 цветов,
3. изображение условно "нельзя" использовать с прозрачным фоном,
4. изображение в других формататх или теряют в качестве, или "весят" больше.

Выбрал расширение gif т.к.:
1. изображение на однородном фоне,
2. изображение содержит менее 256 цветов,
3. изображение можно использовать с прозрачным фоном,
4. наименьший размер файла без потери качества.
